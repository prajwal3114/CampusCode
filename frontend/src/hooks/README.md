# Custom React Hooks

Shared custom hooks for common functionality.

## Hooks to Add

- `useAuth.ts` - Authentication state and functions
- `useWebSocket.ts` - WebSocket connection management
- `useDebounce.ts` - Debounce hook for inputs
- `useLocalStorage.ts` - Local storage state management
- `usePagination.ts` - Pagination logic
- `useInfiniteScroll.ts` - Infinite scroll functionality
- `index.ts` - Hook exports
