# API Services

HTTP client and API endpoint service functions.

## Services to Add

- `api.ts` - Axios base configuration with interceptors
- `auth.service.ts` - Auth API calls (login, register, refresh)
- `users.service.ts` - User API calls
- `competitions.service.ts` - Competition API calls
- `leaderboards.service.ts` - Leaderboard API calls
- `resources.service.ts` - Resources API calls
- `roadmaps.service.ts` - Roadmaps API calls
- `chat.service.ts` - Chat history API calls
