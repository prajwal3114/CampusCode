# Chat Module

Real-time division-wise community chat interface.

## Components

- `ChatWindow.tsx` - Main chat window layout
- `MessageList.tsx` - Scrollable message list
- `MessageInput.tsx` - Message input with send button
- `MessageBubble.tsx` - Individual message display
- `DivisionSelector.tsx` - Select chat room/division
- `OnlineUsers.tsx` - Online user list
- `TypingIndicator.tsx` - Typing indicator

## Pages

- `ChatPage.tsx` - Main chat page

## Hooks

- `useChat.ts` - WebSocket connection and messaging
- `useWebSocket.ts` - Generic WebSocket hook

## Types

- `types.ts` - Chat message and room interfaces
