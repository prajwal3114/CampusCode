# Competitions Module

Competition browsing and management interface.

## Components

- `CompetitionCard.tsx` - Competition card display
- `CompetitionList.tsx` - List of competitions
- `CompetitionDetail.tsx` - Detailed competition view
- `CreateCompetition.tsx` - Competition creation form
- `CompetitionFilters.tsx` - Filter competitions

## Pages

- `CompetitionsPage.tsx` - Browse competitions
- `CompetitionDetailPage.tsx` - Single competition details
- `CreateCompetitionPage.tsx` - Create new competition (organizer)
- `MyCompetitionsPage.tsx` - User's registered competitions

## Hooks

- `useCompetitions.ts` - Fetch and manage competitions

## Types

- `types.ts` - Competition-related interfaces
