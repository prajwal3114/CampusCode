# Roadmaps Module

Structured learning path system interface.

## Components

- `RoadmapCard.tsx` - Roadmap card display
- `RoadmapTimeline.tsx` - Visual timeline of milestones
- `RoadmapProgress.tsx` - Progress tracker
- `MilestoneCard.tsx` - Individual milestone display

## Pages

- `RoadmapsPage.tsx` - Browse roadmaps
- `RoadmapDetailPage.tsx` - Roadmap details with milestones
- `MyRoadmapsPage.tsx` - Enrolled roadmaps

## Hooks

- `useRoadmaps.ts` - Fetch and manage roadmaps

## Types

- `types.ts` - Roadmap and milestone interfaces
