# Resources Module

Learning resources browsing and management.

## Components

- `ResourceCard.tsx` - Resource card display
- `ResourceList.tsx` - List of resources
- `ResourceFilter.tsx` - Filter by domain/type/difficulty
- `ResourceDetail.tsx` - Detailed resource view

## Pages

- `ResourcesPage.tsx` - Browse resources
- `ResourceDetailPage.tsx` - Single resource details
- `MyResourcesPage.tsx` - Bookmarked resources

## Hooks

- `useResources.ts` - Fetch and manage resources

## Types

- `types.ts` - Resource interfaces
