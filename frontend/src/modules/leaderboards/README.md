# Leaderboards Module

Multi-type leaderboard display system.

## Components

- `LeaderboardTable.tsx` - Generic leaderboard table
- `CompetitionLeaderboard.tsx` - Competition-specific rankings
- `CompletionLeaderboard.tsx` - Task completion rankings
- `DomainLeaderboard.tsx` - Domain-wise rankings
- `LeaderboardFilters.tsx` - Filter options

## Pages

- `LeaderboardsPage.tsx` - Main leaderboards page with tabs

## Hooks

- `useLeaderboards.ts` - Fetch leaderboard data

## Types

- `types.ts` - Leaderboard interfaces

## Leaderboard Types

1. Competition - Rankings within competitions
2. Completion - Based on tasks/challenges completed
3. Domain - Tech domain rankings (frontend, backend, AI, etc.)
