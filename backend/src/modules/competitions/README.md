# Competitions Module

Competition creation and management system.

## Structure

- `competitions.controller.ts` - Competition CRUD
- `competitions.service.ts` - Business logic
- `competitions.repository.ts` - Database operations
- `competitions.routes.ts` - Route definitions
- `competitions.dto.ts` - Validation schemas
- `competitions.types.ts` - TypeScript interfaces
- `competitions.test.ts` - Unit tests

## Features to Implement

- Create competition (organizer/admin)
- List competitions (public/private)
- Competition details
- Update competition
- Delete competition
- Register for competition
- Submit competition entry
- Competition status management (draft, active, closed)
