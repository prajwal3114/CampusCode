# Resources Module

Learning resources management system.

## Structure

- `resources.controller.ts` - Resource CRUD
- `resources.service.ts` - Business logic
- `resources.repository.ts` - Database operations
- `resources.routes.ts` - Route definitions
- `resources.dto.ts` - Validation schemas
- `resources.types.ts` - TypeScript interfaces
- `resources.test.ts` - Unit tests

## Features to Implement

- Create resource (organizer/admin)
- List resources (filtered by domain, type, difficulty)
- Resource details

- Delete resource
- Bookmark resources
- Resource completion tracking
- Resource ratings/reviews
