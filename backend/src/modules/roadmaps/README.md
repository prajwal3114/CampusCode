# Roadmaps Module

Structured learning path system.

## Structure

- `roadmaps.controller.ts` - Roadmap CRUD
- `roadmaps.service.ts` - Business logic
- `roadmaps.repository.ts` - Database operations
- `roadmaps.routes.ts` - Route definitions
- `roadmaps.dto.ts` - Validation schemas
- `roadmaps.types.ts` - TypeScript interfaces
- `roadmaps.test.ts` - Unit tests
