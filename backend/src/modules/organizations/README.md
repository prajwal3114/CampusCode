# Organizations Module

Multi-tenant organization/college management.

## Structure

- `organizations.controller.ts` - Organization CRUD
- `organizations.service.ts` - Business logic
- `organizations.repository.ts` - Database operations
- `organizations.routes.ts` - Route definitions
- `organizations.dto.ts` - Validation schemas
- `organizations.types.ts` - TypeScript interfaces
- `organizations.test.ts` - Unit tests
