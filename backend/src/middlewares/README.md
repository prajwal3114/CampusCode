# Express Middlewares

Common middlewares for authentication, authorization, validation, and error handling.

## Files to Add

- `auth.middleware.ts` - JWT authentication middleware
- `rbac.middleware.ts` - Role-based access control (Admin/Organizer/User)
- `rate-limit.middleware.ts` - Rate limiting with Redis
- `validate.middleware.ts` - Request validation using Zod
- `error-handler.ts` - Global error handler
- `not-found.ts` - 404 handler
